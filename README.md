## Run Locally

Clone the project

```bash
  git clone https://github.com/sjsouvik/react-components-hooks.git
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```

Now, you can visit `http://localhost:5173/` to see the project running locally
